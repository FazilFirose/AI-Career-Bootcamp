# Smart Calculator

A menu-driven calculator built in Python as part of my AI Career Bootcamp.

## Version 1.0
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero protection
- Continuous menu until exit
## Version 2.0 Features

- Refactored code using functions
- Input validation with try-except
- Reusable number input function
- Calculation history saved to file
- View history
- Clear history
- Improved menu and code structure
